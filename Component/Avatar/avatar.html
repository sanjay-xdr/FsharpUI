Avatar


Example of Image Avatar


Avatar can be used to show user's profile picture on profile information page, on navigation bar, in blogs grid items.

Avatar is available in 4 different sizes. You can use image in Avatar. You need to include class avatar and for size add class according to size avatar-xl, avatar-lg, avatar-md, avatar-sm (e.g. class="round-img avatar-lg")
